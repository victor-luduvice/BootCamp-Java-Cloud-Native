- **Java**
- **Spring Boot**
- **Microsserviços**
- **Docker e Kubernetes**
- **Cloud Computing (AWS, Azure, Google Cloud)**
- **Banco de Dados SQL e NoSQL**
- **CI/CD e DevOps**
- **APIs REST e Segurança**

## 🛠 Como Executar os Projetos

1. **Clone o repositório**
   ```sh
   git clone https://github.com/victor-luduvice/BootCamp-Java-Cloud-Native.git
   ```

2. **Acesse o diretório do projeto desejado**
   ```sh
   cd BootCamp-Java-Cloud-Native/modulo-1
   ```

3. **Execute o projeto (Exemplo com Spring Boot)**
   ```sh
   mvn spring-boot:run
   ```

4. **Acesse a aplicação no navegador ou via API**
   ```
   http://localhost:8080
   ```

## 🎯 Objetivo do Bootcamp

O **Bootcamp Bradesco - Java Cloud Native** tem como objetivo capacitar desenvolvedores a construir aplicações escaláveis e modernas na nuvem. Durante o curso, são abordados conceitos como **microsserviços, segurança, arquitetura de software, DevOps e integração contínua**.
